# Sample API

#### Endpoints
- https://sample-api.k8s.joshuatonga.com/api/jokes
- https://sample-api.k8s.joshuatonga.com/healthcheck

## Prerequisites
- Install [Node.js](https://nodejs.org/en/) v16.14.2
- npm v8.5.0
- docker

## Getting started
```
npm install
npm start
```

## Building
```
docker build joshuatonga/sample-nodejs-api:latest .
```
